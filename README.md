# TicTacToe (iOS – SwiftUI)

A simple 2-player Tic-Tac-Toe game built for iOS.

## Demo


## Features
- Two players on same device (X and O)
- 3×3 grid UI
- Tracks current player
- Win detection
- Draw detection
- Restart button (X or O starts)
- Alert popup for winner/draw
- Smooth animations

## Project Structure
- `ContentView.swift` – UI + interaction
- `TicTacToeViewModel.swift` – game logic
- `Models.swift` – Player enum
- `TicTacToeApp.swift` – app entry point

## How to Run
1. Clone this repo:
```
git clone https://github.com/Nic815/TicTacToe.git
```
3. Open `TicTacToe.xcodeproj` in Xcode  
4. Run on Simulator or physical device

## Requirements
- Xcode 14+ or 15+
- iOS 14+ device or simulator
